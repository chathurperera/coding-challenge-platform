# coding-challenge-platform

#### Env variables:

Create .env file in backend directory and add the following:

```
MONGO_URI = "Mongo URI"
JWT_SECRET = "JWT secret"
JWT_REFRESH_SECRET = "Refresh token secret"
```

#### Frontend:

```
cd frontend
npm install
npm run dev
```

#### Backend:

```
cd backend
npm install
npm start
```
